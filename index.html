Key Concepts used

HTML Structure

The HTML file (index.html) defines the structure of the webpage. It uses HTML5 semantic elements like <header>, <section>, and <footer> to organize content.

CSS Styling

The styling is managed using an external CSS file (main.css). It defines the visual appearance, layout, and colors of the elements on the page.

JavaScript

This script (main.js) listens for the form submission event and prevents the default form submission behavior. It then retrieves the values from the form fields (name, email, and message) and logs them to the console. You can customize this script based on your specific requirements and add more functionality as needed.
